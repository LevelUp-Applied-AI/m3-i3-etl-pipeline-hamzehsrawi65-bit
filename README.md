## Project Overview
This project is designed to prepare a collaborative Python environment for analyzing hospital admission records. The goal is to help contributors quickly set up the repository, understand the project structure, and begin building analysis workflows and reusable source code for hospital admission data.

## Data Sources
This project uses hospital admission records data for analysis and future reporting workflows.

Data is not tracked in this repository. See the setup instructions below
for how to obtain and place the data files before running any analysis.

Expected raw data location:
- `data/raw/admissions.csv`
